# 高效阅读技巧

学习如何高效地阅读学术论文并创建有价值的标注。

## 分层阅读法

### 第一遍：快速浏览

1. 阅读标题、摘要、结论
2. 查看图表和标题
3. 浏览社区高赞标注
4. 判断是否值得深入阅读

### 第二遍：精读重点

1. 仔细阅读 Introduction 和 Methods
2. 理解研究问题和方法
3. 创建标注记录关键信息
4. 查看他人对重点部分的标注

### 第三遍：批判性阅读

1. 评估论文的创新性和局限性
2. 与已有研究对比
3. 思考如何应用到自己的研究
4. 参与社区讨论

### 笔记模板

创建标注时可以参考以下模板：

```markdown
## 核心观点
[总结论文的主要贡献]

## 方法亮点
[记录创新的方法或技术]

## 疑问
[记录不理解或质疑的地方]

## 启发
[对自己研究的启发]
```

## 利用社区智慧

### 学习他人标注

- 查看点赞最多的标注
- 关注领域专家的标注
- 参考不同视角的解读

### 参与讨论

- 对有价值的标注点赞
- 回复和补充观点
- 提出问题引发讨论
