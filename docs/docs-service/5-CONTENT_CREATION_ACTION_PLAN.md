#### 上午：自我优化

### Day 6：视频制作

#### 方案A：简单录屏（推荐新手）

**工具：**
- OBS Studio（免费录屏）
- 手机摄像头（真人出镜）
- 剪映（免费剪辑）

**流程：**
1. 录制真人口述（对着手机讲）
2. 录制屏幕操作（展示Researchopia）
3. 剪映导入素材，按脚本剪辑
4. 添加字幕、背景音乐
5. 导出视频

**时间分配：**
- 录制：2小时
- 剪辑：3-4小时
- 全天完成5条视频

#### 方案B：PPT式视频（最简单）

**工具：**
- PPT/Keynote
- 录屏软件

**流程：**
1. 制作10-15页PPT（每条视频2-3页）
2. 插入图片、文字
3. 录制PPT演示 + 语音讲解
4. 简单剪辑

**优势：**
- 不需要真人出镜（降低心理门槛）
- 制作快速
- 适合第一次尝试
