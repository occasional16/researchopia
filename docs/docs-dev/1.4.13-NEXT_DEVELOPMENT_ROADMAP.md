## 📋 下一步开发方向（优先级排序）

### 🔥 P0 - 核心体验优化（1-2周）

#### 1. 实时协作增强
- [ ] **标注实时同步**（WebSocket）
  - 当前：刷新加载
  - 目标：多人同时标注实时显示
  - 技术：Supabase Realtime

- [ ] **在线用户列表**
  - 显示当前阅读同一论文的用户
  - 在线状态指示器
  - 可快速@用户

#### 2. 标注交互深化
- [ ] **标注回复嵌套**
  - 支持评论的评论
  - @提及功能
  - 通知系统

- [ ] **标注版本历史**
  - 查看标注编辑记录
  - 支持恢复旧版本

### ⚡ P1 - 生产力工具（2-3周）

#### 3. PDF阅读增强
- [ ] **标注分组管理**
  - 创建标注集合
  - 按主题/章节分组
  - 导出为笔记

- [ ] **AI辅助标注**
  - 自动摘要选中段落
  - 翻译功能
  - 相关文献推荐

#### 4. 会话功能完善
- [ ] **会话管理面板**
  - 创建/加入会话（当前主按钮功能）
  - 会话列表与状态
  - 会话权限管理

- [ ] **会话统计仪表板**
  - 参与成员贡献度
  - 标注热点图
  - 阅读进度追踪

### 🎨 P2 - 体验优化（1-2周）

#### 5. UI/UX改进
- [ ] **暗色模式完整支持**
  - 当前：部分适配
  - 目标：全面跟随Zotero主题

- [ ] **快捷键系统**
  - 快速标注：Ctrl+H（高亮）、Ctrl+N（笔记）
  - 快速筛选：F1（全部）、F2（关注）
  - 快速评论：Ctrl+Enter

- [ ] **标注样式定制**
  - 自定义高亮颜色
  - 标注图标库
  - 个性化用户名标签

#### 6. 性能优化
- [ ] **大文档优化**
  - 虚拟滚动（100+标注）
  - 懒加载评论
  - 图片缓存策略

- [ ] **离线支持**
  - 离线标注队列
  - 网络恢复自动同步

### 🔮 P3 - 高级功能（3-4周）

#### 7. 团队协作
- [ ] **读书会功能**
  - 定时会话（例：每周三8点）
  - 会议记录
  - 任务分配

- [ ] **共享笔记本**
  - 团队共享标注库
  - 权限管理（只读/编辑/管理员）

#### 8. 数据分析
- [ ] **个人阅读统计**
  - 阅读时长
  - 标注数量
  - 高频词云

- [ ] **论文影响力分析**
  - 被标注次数排行
  - 热门段落识别

## 💡 创新方向探索

### AI集成
- **智能标注推荐**：基于用户历史推荐相似论文的标注
- **自动生成摘要**：将标注转为结构化笔记
- **问答机器人**：基于论文和标注回答问题

### 多平台扩展
- **Web版查看器**：不依赖Zotero的浏览器端查看
- **移动端应用**：iOS/Android离线阅读+同步
- **VS Code扩展**：代码注释与论文标注联动
