# 共读会话

共读会话是 Researchopia 的核心功能，允许多人实时协作阅读同一篇论文。

## 创建会话

1. 在 Zotero 中选择论文
2. 点击 **"Reading Session"** 标签
3. 点击 **"Create Session"**
4. 选择会话类型：
   - **Public**: 任何人可加入
   - **Private**: 需要邀请码

## 加入会话

### 使用邀请码

1. 点击 **"Join Session"**
2. 输入 6 位邀请码
3. 点击 **"Join"**

### 浏览公开会话

1. 点击 **"Browse Public Sessions"**
2. 查看活跃会话列表
3. 点击 **"Join"** 加入

## 会话功能

### 实时标注

会话成员创建的标注会实时同步给所有参与者。

### 即时聊天

在会话中与其他成员文字聊天，讨论论文内容。

### 成员管理

查看在线和离线成员，了解谁在阅读。
