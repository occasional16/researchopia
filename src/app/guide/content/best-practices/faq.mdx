# 常见问题 FAQ

## 通用问题

### Researchopia 是免费的吗？

**是的！** Researchopia 目前完全免费使用，包括：
- 无限制搜索和浏览论文
- 无限制创建标注
- 无限制共读会话
- 所有社交功能

未来可能推出高级功能，但核心功能将始终免费。

### 如何在 Researchopia 上找到论文全文？

Researchopia 不提供论文 PDF 托管，但会链接到：
- 期刊官网
- 开放获取仓库（arXiv, PubMed Central）
- 作者个人主页

建议通过学校图书馆或 Google Scholar 访问。

### 我的标注会被其他人看到吗？

取决于您的隐私设置：
- **公开**: 所有用户可见
- **仅好友**: 只有关注的人可见
- **私密**: 只有您自己可见

默认为"公开"，可在设置中修改。

## 网站相关

### 为什么搜索结果为空？

可能的原因：
- 关键词拼写错误
- 筛选条件太严格
- 论文尚未被索引

### 如何搜索中文论文？

直接输入中文关键词即可，系统支持多语言搜索。

## Zotero 插件

### 插件支持哪些版本？

- ✅ Zotero 7.0+
- ✅ Zotero 8.0 Beta（推荐）
- ❌ Zotero 6.x 及以下（不支持）

### 插件无法安装怎么办？

1. 检查 Zotero 版本
2. 尝试从不同来源下载 .xpi
3. 查看控制台错误日志

### 标注不同步怎么办？

1. 检查是否已登录
2. 确认同步设置
3. 手动点击同步按钮
4. 检查网络连接

## 浏览器扩展

### 为什么没有检测到 DOI？

可能的原因：
1. 网站未在支持列表中
2. 论文没有 DOI
3. 扩展未正确加载

### 支持哪些浏览器？

- ✅ Chrome (v88+)
- ✅ Edge (v88+)
- ✅ Brave
- ⏳ Firefox (即将支持)

## 技术问题

### 如何引用 Researchopia 中的标注？

如果在论文中使用社区标注的观点：

```
Smith, J. (2025). 标注于 "Deep Learning for X". 
Researchopia. https://www.researchopia.com/papers/10.xxxx/...
```

### Researchopia 与 Zotero 官方的关系？

Researchopia 是独立的第三方项目，与 Zotero 官方无关。
我们基于 Zotero 的开放 API 开发，遵循 AGPL-3.0 许可证。

### 数据存储在哪里？

- **后端数据库**: Supabase (AWS)
- **前端网站**: Vercel (全球 CDN)
- 数据传输使用 HTTPS 加密
