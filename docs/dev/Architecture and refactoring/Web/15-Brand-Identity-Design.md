# Researchopia 品牌标识设计建议

## 当前品牌标语

| 类型 | 中文 | 英文 |
|------|------|------|
| **品牌名** | 研学港 | Researchopia |
| **主标语** | 研学并进，智慧共享 | Where Research Meets Community |
| **副标语** | 开放的学术交流和共享平台 | An open academic community |

---

## 关于"学术→知识"的改动分析

### 优势 ✅
| 维度 | 说明 |
|------|------|
| **覆盖面更广** | "知识"包含学术论文、网页、视频、课程等多种载体 |
| **契合知识网功能** | 新开发的"网页评价"功能不限于学术论文 |
| **降低门槛感** | "知识"比"学术"更亲民，吸引更多非学术用户 |

### 风险 ⚠️
| 维度 | 说明 |
|------|------|
| **定位模糊** | 可能与知乎、得到等知识平台混淆 |
| **品牌稀释** | 失去"学术"的专业调性，核心用户可能流失 |
| **名称冲突** | "Researchopia"中的"Research"指向学术研究 |

---

## 建议方案

### 方案 A：保持"学术"，副标语扩展 ⭐推荐

```
主标语：研学并进，智慧共享
副标语：开放的学术交流与知识共享平台
英文：Open platform for academic exchange and knowledge sharing
```

**理由**：保持学术定位的同时，用"知识共享"暗示功能扩展

### 方案 B：改为"知识"

```
副标语：开放的知识交流和共享平台
英文：Open platform for knowledge exchange and sharing
```

**风险**：需同步考虑是否改品牌名（如"知学港"）

### 方案 C：双轨定位

```
对内/学术用户：开放的学术交流和共享平台
对外/普通用户：构建你的知识网络
```

**适用**：不同场景使用不同标语（如浏览器扩展已用"构建你的知识网络"）

## 品牌名称分析

### "研学港"拆解
- **研** = 研究 (Research)
- **学** = 学习/学术 (Learning/Academic)  
- **港** = 港湾/聚集地 (Harbor/Hub)

### 英文名"Researchopia"拆解
- **Research** = 研究
- **-opia** = 理想国（源自 Utopia）

**结论**：中英文品牌名都强调"研究/学术"，如果改为"知识"定位，可能需要考虑品牌名变更（成本较高）。

---

## 最终建议

| 决策点 | 建议 |
|--------|------|
| **品牌名** | 保持"研学港/Researchopia"不变 |
| **主标语** | 保持"研学并进，智慧共享" |
| **副标语** | 采用方案 A："开放的学术交流与知识共享平台" |
| **浏览器扩展** | 可保持"构建你的知识网络"（场景化差异） |
