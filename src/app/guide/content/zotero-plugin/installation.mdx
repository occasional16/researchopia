# 安装和配置 Zotero 插件

将 Researchopia 集成到 Zotero 文献管理器中，享受无缝的标注同步和协作阅读体验。

## 系统要求

### 前置条件

在安装插件前，请确保：

### 检查 Zotero 版本

1. 打开 Zotero
2. 点击 **帮助** → **关于 Zotero**
3. 查看版本号

如果版本低于 7.0，请先升级 Zotero。

## 安装插件

### 方法 1: 从官网下载（推荐）

1. 访问 [researchopia.com/plugin](https://researchopia.com/plugin)
2. 点击 **"下载 Zotero 插件"** 按钮
3. 下载 `researchopia-v0.3.3.xpi` 文件

### 方法 2: 从 GitHub 下载

1. 访问 [GitHub Releases](https://github.com/occasional16/researchopia/releases)
2. 找到最新版本
3. 下载 Assets 中的 `.xpi` 文件

### 验证安装

安装成功后，您应该看到：

- ✅ 工具栏出现 Researchopia 图标 🌐
- ✅ 工具菜单中有 "Researchopia" 选项
- ✅ 右侧面板有新的标签页

## 首次配置

### 登录账号

1. 点击工具栏的 Researchopia 图标
2. 或点击 **工具** → **Researchopia** → **Login**
3. 在弹出窗口输入您的邮箱和密码
4. 点击 **"登录"**

### 配置同步选项

打开设置：**工具** → **Researchopia** → **Settings**

#### 标注同步设置

```
✅ 自动同步: 创建标注后立即同步到云端
⏸️ 手动同步: 需要手动点击同步按钮
❌ 不同步: 标注只保存在本地
```

#### 隐私设置

选择标注的默认可见性：

- 🌍 **公开**: 所有人可见
- 👥 **仅好友**: 只有关注的人可见
- 🔒 **私密**: 只有自己可见

#### 通知设置

- 新评论通知
- 共读会话邀请通知
- 新关注者通知

## 更新插件

### 自动更新

Zotero 会自动检查插件更新。当有新版本时：

1. 附加组件管理器会显示更新通知
2. 点击 **"立即更新"**
3. 重启 Zotero

### 手动更新

1. 下载最新版本的 `.xpi` 文件
2. 按照安装步骤重新安装
3. 覆盖旧版本

## 卸载插件

如需卸载：

1. 工具 → 附加组件
2. 找到 Researchopia
3. 点击 **"移除"**
4. 重启 Zotero

## 故障排除

### 插件无法安装

**问题**: 提示 "无法安装附加组件"

**解决方案**:
- 检查 Zotero 版本是否 ≥ 7.0
- 尝试从不同来源下载 .xpi 文件
- 检查文件是否完整（约 500KB）

### 无法登录

**问题**: 输入邮箱密码后无反应

**解决方案**:
- 检查网络连接
- 确认邮箱和密码正确
- 尝试重置密码
- 查看 Zotero 控制台错误（帮助 → 开发者 → 控制台）

### 标注不同步

**问题**: 创建的标注没有上传到云端

**解决方案**:
- 检查是否已登录
- 确认同步设置不是"不同步"
- 手动点击同步按钮
- 查看网络连接状态

## 常见问题

### 插件是免费的吗？

是的，完全免费且开源！

### 支持哪些语言？

目前支持中文和英文，更多语言即将推出。

### 会影响 Zotero 性能吗？

不会。插件经过优化，对 Zotero 性能影响极小。
