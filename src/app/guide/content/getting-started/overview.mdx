# Researchopia 平台概览

研学港 Researchopia 是一个开放的学术交流与知识共享平台，旨在打破学术壁垒，促进研究者之间的协作与知识共享。

## 核心功能

### 1. 🔍 论文搜索与发现

- **智能搜索**: 支持关键词、作者、DOI 等多种搜索方式
- **高级筛选**: 按年份、期刊、学科领域等条件筛选
- **个性化推荐**: 基于阅读历史的智能推荐系统

### 2. 💬 社区标注与讨论

- **公开标注**: 在论文上创建和分享您的标注
- **互动评论**: 与其他研究者讨论论文内容
- **点赞系统**: 识别有价值的标注和观点

### 3. 👥 实时共读会话

- **协作阅读**: 与同事一起实时阅读同一篇论文
- **即时聊天**: 在阅读过程中进行文字交流
- **会话管理**: 创建公开或私密的阅读会话

### 4. 📱 多平台支持

- **网站**: 功能完整的 Web 应用
- **Zotero 插件**: 集成到文献管理器
- **浏览器扩展**: 在任何网页快速访问

## 工作流程示例

### 场景 1: 日常文献阅读

```
1. 在 Google Scholar 搜索论文
   ↓
2. 浏览器扩展自动识别 DOI
   ↓
3. 点击悬浮图标查看社区标注
   ↓
4. 在 Zotero 中打开论文
   ↓
5. 创建个人标注并同步
```

### 场景 2: 团队协作阅读

```
1. 教师在 Zotero 中选择论文
   ↓
2. 创建私密共读会话
   ↓
3. 分享邀请码给学生
   ↓
4. 学生加入会话
   ↓
5. 实时讨论论文内容
```

### 场景 3: 学术社交

```
1. 在 Researchopia 网站搜索论文
   ↓
2. 查看热门标注
   ↓
3. 关注感兴趣的研究者
   ↓
4. 点赞和评论标注
   ↓
5. 接收关注者动态通知
```
