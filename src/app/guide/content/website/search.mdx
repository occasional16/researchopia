# 搜索和浏览论文

Researchopia 提供强大的搜索功能，帮助您快速找到感兴趣的学术论文。

## 基本搜索

### 使用搜索框

1. 在首页顶部找到搜索框
2. 输入关键词（如 "machine learning"）
3. 按 Enter 键或点击搜索图标
4. 浏览搜索结果

### 搜索技巧

#### 精确匹配

使用引号搜索精确短语：

```
"deep learning"
```

#### 排除关键词

使用减号排除不需要的结果：

```
artificial intelligence -robotics
```

#### 按作者搜索

搜索特定作者的论文：

```
author:"Andrew Ng"
```

#### DOI 搜索

直接输入 DOI 跳转到论文：

```
10.1038/nature12345
```

## 高级筛选

### 筛选条件

使用左侧面板的筛选选项：

- **发表年份**: 2020-2025
- **期刊/会议**: Nature, Science, NeurIPS...
- **学科领域**: 计算机科学, 生物学, 物理学...
- **开放获取**: 仅显示免费论文

## 浏览推荐

### 个性化推荐

首页提供基于您阅读历史的推荐：

- **热门论文**: 近期浏览和标注最多的论文
- **最新发表**: 各领域最新论文
- **相关推荐**: 根据阅读历史推荐
- **关注的作者**: 关注研究者的新作

### 领域探索

按学科浏览论文：

1. 点击"领域"标签
2. 选择感兴趣的学科
3. 查看该领域的热门和最新论文

## 搜索结果页面

### 论文卡片信息

每个搜索结果包含：

- **标题**: 点击查看详情
- **作者**: 点击查看作者其他论文
- **摘要**: 快速了解论文内容
- **发表信息**: 期刊/会议、年份
- **标注数**: 社区标注数量
- **快速操作**: 收藏、分享、导出

### 快速预览

悬停在论文卡片上可以看到：

- 完整摘要
- 引用数和 h-index
- 标注热度图
- 快速收藏按钮

### 查看历史

1. 点击搜索框
2. 下拉菜单显示最近搜索
3. 点击快速执行搜索

### 清除历史

- 个人设置 → 隐私 → 清除搜索历史

## 保存搜索

### 创建搜索订阅

对于重要的搜索查询，您可以：

1. 执行搜索
2. 点击"保存搜索"
3. 设置通知频率（每日/每周）
4. 当有新论文匹配时接收邮件通知

## 常见问题

### 为什么搜索结果为空？

可能的原因：
- 关键词拼写错误
- 筛选条件太严格
- 论文尚未被索引

### 如何搜索中文论文？

直接输入中文关键词即可，系统支持多语言搜索。

### 可以按照影响因子筛选吗？

目前不支持直接按影响因子筛选，但可以按期刊名称筛选。
