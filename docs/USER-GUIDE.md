# 第一部分:网站使用

<a name="账号注册和登录"></a>
## 1. 账号注册和登录

### 1.1 注册新账号

1. 访问 [www.researchopia.com](https://www.researchopia.com)
2. 点击右上角 **"注册"** 按钮
3. 填写注册信息:
   - **邮箱地址**: 用于接收验证邮件和通知
   - **密码**: 至少8位,建议包含字母和数字
   - **用户名**: 显示在个人主页和评论中
4. 点击 **"创建账号"**
5. 查收邮箱中的验证邮件,点击链接激活账号

**💡 提示**: 建议使用学校或机构邮箱注册,方便后续验证学术身份

### 1.2 登录账号

1. 点击右上角 **"登录"** 按钮
2. 输入邮箱和密码
3. (可选)勾选 **"记住我"** 保持登录状态
4. 点击 **"登录"**

**忘记密码?**
1. 点击登录框下方的 **"忘记密码"**
2. 输入注册邮箱
3. 查收重置密码邮件
4. 点击邮件中的链接设置新密码

<a name="搜索和浏览论文"></a>
## 2. 搜索和浏览论文

### 2.1 使用搜索功能

**基本搜索**:
1. 在首页顶部搜索框输入关键词
2. 按 Enter 或点击搜索图标
3. 浏览搜索结果

**搜索技巧**:
```
# 精确匹配
"machine learning"

# 排除词
artificial intelligence -robotics

# 作者搜索
author:"John Smith"

# DOI搜索
10.1038/nature12345
```

### 2.2 筛选和排序

**筛选条件**:
- **发表年份**: 2020-2025
- **期刊/会议**: Nature, Science, IEEE...
- **学科领域**: 计算机科学, 生物学, 物理学...
- **开放获取**: 仅显示免费论文

**排序方式**:
- **相关性**: 最匹配搜索关键词
- **发表时间**: 最新论文优先
- **引用数**: 高影响力论文优先
- **热度**: 平台内浏览和标注最多

### 2.3 浏览推荐论文

首页提供个性化推荐:
- **热门论文**: 近期浏览和标注最多的论文
- **最新发表**: 各领域最新论文
- **相关推荐**: 根据您的阅读历史推荐
- **关注的作者**: 您关注的研究者的新作

<a name="查看论文详情"></a>
## 3. 查看论文详情

点击任意论文标题进入详情页:

### 3.1 论文信息

- **标题和摘要**: 快速了解论文内容
- **作者列表**: 点击作者名可查看其他论文
- **发表信息**: 期刊/会议, 卷期, 页码, DOI
- **引用格式**: 提供多种引用格式(APA, MLA, BibTeX)

### 3.2 社区标注

查看其他用户的标注和评论:
- **高亮标注**: 重要段落和句子
- **笔记评论**: 用户对内容的理解和思考
- **点赞和回复**: 与其他读者互动

**按热度排序**: 点赞最多的标注排在前面  
**按时间排序**: 最新标注排在前面  
**按页码排序**: 按论文页码顺序显示

### 3.3 快速操作

- **收藏论文**: 点击 ⭐ 图标添加到收藏夹
- **分享论文**: 复制链接或分享到社交媒体
- **导出引用**: 下载 BibTeX, RIS, EndNote 格式
- **在Zotero中打开**: 如果已安装插件,直接在Zotero中打开

# 第二部分:Zotero插件

<a name="安装zotero插件"></a>
## 6. 安装Zotero插件

### 6.1 前置要求

- **Zotero**: 版本 7.0 或更高(推荐 8.0 Beta)
- **操作系统**: Windows, macOS, Linux
- **网络**: 需要稳定的互联网连接

### 6.2 安装步骤

**方法一:从官方网站安装**
1. 访问 [www.researchopia.com/plugin](https://www.researchopia.com/plugin)
2. 点击 **"下载Zotero插件"**
3. 下载 `researchopia.xpi` 文件
4. 打开 Zotero → 工具 → 附加组件
5. 点击齿轮图标 → Install Add-on From File
6. 选择下载的 `researchopia.xpi` 文件
7. 点击 **"Install"**
8. 重启 Zotero

**方法二:从GitHub安装**(开发版)
1. 访问 [GitHub Releases](https://github.com/occasional16/researchopia/releases)
2. 下载最新版本的 `.xpi` 文件
3. 按照方法一的步骤4-8安装

### 6.3 验证安装

安装成功后:
- 工具栏应出现 Researchopia 图标
- 工具菜单中出现 "Researchopia" 选项
- 右侧面板出现 "Community Annotations" 和 "Reading Session" 标签

## 8. 查看社区标注

### 8.1 访问社区标注

1. 在 Zotero 文献库中选择一篇论文(必须有DOI)

3. 插件会自动加载该论文的社区标注

每条标注包含:

### 8.3 交互操作

**点赞标注**:
- 点击标注下方的 ❤️ 图标
- 再次点击取消点赞

3. 点击 **"发送"**

- 点击标注可在 Researchopia 网站上查看完整讨论

### 8.4 筛选和排序

**筛选**:
- **所有标注**: 显示所有用户的标注

**排序**:
- **按热度**: 点赞最多的标注优先

<a name="创建共读会话"></a>
## 9. 创建共读会话

### 9.1 什么是共读会话

共读会话是实时协作阅读功能,允许多人同时阅读同一篇论文,分享标注和讨论。

**应用场景**:
- 📚 课程讨论: 教师带领学生共同阅读文献
- 🤝 团队协作: 课题组成员一起研读论文
- 🎓 读书会: 与同行定期共读经典论文
- 💡 学术交流: 与全球研究者实时讨论

### 9.2 创建会话步骤

1. 在 Zotero 中选择一篇论文(必须有DOI)
2. 点击右侧面板的 **"Reading Session"** 标签
3. 点击 **"Create Session"** 按钮
4. 选择会话类型:
   - **Public (公开)**: 任何人都可以加入
   - **Private (私密)**: 需要邀请码才能加入
5. 点击 **"Create"**

### 9.3 会话信息

创建成功后显示:
- **会话ID**: 唯一标识符
- **邀请码**: 6位字母数字(私密会话)
- **成员列表**: 当前参与的用户
- **在线状态**: 谁在线阅读

### 9.4 分享邀请码

**私密会话**:
1. 复制邀请码(例如: `ABC123`)
2. 通过邮件、微信、Slack 等发送给同事
3. 对方使用邀请码加入会话

**公开会话**:
- 自动出现在公开会话列表
- 任何人都可以直接加入

<a name="加入共读会话"></a>
## 10. 加入共读会话

### 10.1 使用邀请码加入

1. 点击 **"Join Session"** 按钮
2. 输入6位邀请码(例如: `ABC123`)
3. 点击 **"Join"**
4. 成功加入后会显示会话信息

### 10.2 浏览公开会话

1. 点击 **"Browse Public Sessions"**
2. 查看当前活跃的公开会话列表:
   - 论文标题
   - 创建者
   - 参与人数
   - 创建时间
3. 点击 **"Join"** 加入感兴趣的会话

### 10.3 会话内功能

加入会话后:
- **查看实时标注**: 其他成员创建的标注会实时显示
- **查看成员**: 查看当前在线和离线的成员
- **实时聊天**: 与会话成员文字聊天
- **创建标注**: 您的标注会同步给所有成员

---

<a name="实时聊天"></a>
## 11. 实时聊天

### 11.1 发送消息

1. 在会话面板找到 **"Chat"** 区域
2. 在输入框输入消息
3. 按 Enter 或点击 **"Send"** 发送
4. 消息会实时显示给所有会话成员

### 11.2 消息类型

**普通消息**:
```
大家好,这篇论文的方法很有意思
```

**引用标注**:
```
@标注ID 这个地方的推导有问题吧?
```

**@提及用户**:
```
@JohnSmith 你对这个结论怎么看?
```

### 11.3 聊天记录

- 聊天记录会保存在服务器
- 随时可以查看历史消息
- 支持搜索关键词

<a name="同步个人标注"></a>
## 12. 同步个人标注

### 12.1 自动同步

当您在 Zotero PDF 阅读器中创建标注时:
1. 高亮文本或添加笔记
2. 插件会自动检测到新标注
3. 根据设置决定是否同步到服务器

### 12.2 同步设置

工具 → Researchopia → Settings:

**同步选项**:
- ✅ **自动同步**: 创建标注后立即同步
- ⏸️ **手动同步**: 需要手动点击同步按钮
- ❌ **不同步**: 标注只保存在本地

**隐私设置**:
- 🌍 **公开**: 所有人可见
- 👥 **仅好友**: 只有关注的人可见
- 🔒 **私密**: 只有自己可见

### 12.3 管理标注

**编辑标注**:
1. 在 "My Annotations" 标签中找到标注
2. 点击 **"Edit"**
3. 修改文本或评论
4. 点击 **"Save"**

**删除标注**:
1. 点击标注旁的 **"Delete"** 按钮
2. 确认删除
3. 标注会从服务器移除

<a name="doi自动识别"></a>
## 14. DOI自动识别

### 14.1 支持的网站

扩展会在以下学术网站自动检测DOI:

**主流期刊**:
- Nature (nature.com)
- Science (science.org)
- Cell (cell.com)
- Springer (springer.com)
- Wiley (wiley.com)
- Elsevier (sciencedirect.com)

**学术数据库**:
- IEEE Xplore (ieeexplore.ieee.org)
- ACM Digital Library (dl.acm.org)
- PubMed (pubmed.ncbi.nlm.nih.gov)

**预印本平台**:
- arXiv (arxiv.org)
- bioRxiv (biorxiv.org)
- SSRN (ssrn.com)

### 14.2 识别指示

**检测成功**:
- 页面右侧出现悬浮的 Researchopia 图标
- 图标上有绿色指示点
- 悬停显示 "DOI detected: 10.xxxx/xxxxx"

**未检测到DOI**:
- 图标不显示
- 或图标显示为灰色

### 14.3 手动输入DOI

如果自动检测失败:
1. 点击浏览器工具栏的扩展图标
2. 在弹出窗口输入 DOI
3. 点击 **"Go"**
4. 跳转到 Researchopia 论文页面

<a name="快速访问论文"></a>
## 15. 快速访问论文

### 15.1 悬浮图标

当检测到DOI后:
- 页面右侧出现悬浮图标
- 可以拖拽到任意位置
- 松手后自动吸附到边缘

### 15.2 点击图标

点击悬浮图标有三种行为(可在设置中配置):

**选项1:打开侧边栏**(默认)
- 在当前页面右侧弹出侧边栏
- 加载 Researchopia 论文详情页
- 可以查看社区标注和评论
- 不离开当前期刊网站

**选项2:新标签页打开**
- 在新标签页打开 Researchopia
- 显示完整的论文详情页
- 适合深度阅读和互动

**选项3:在Zotero中打开**
- 如果已安装 Zotero 和插件
- 自动在 Zotero 中打开对应论文
- 直接进入阅读和标注

### 15.3 侧边栏功能

侧边栏中可以:
- 查看论文摘要和基本信息
- 浏览社区标注
- 点赞和评论标注
- 收藏论文
- 分享到社交媒体
- 关闭按钮(右上角 ×)

<a name="自定义设置"></a>
## 16. 自定义设置

### 16.1 打开设置页面

1. 点击浏览器工具栏的扩展图标
2. 点击弹出窗口右上角的 **"Settings"** 齿轮图标
3. 或者 右键扩展图标 → **"Options"**

### 16.2 可配置选项

**显示设置**:
- ✅ 启用悬浮图标
- 🎨 图标颜色主题(浅色/深色)
- 📍 图标默认位置(左/右)
- 📏 侧边栏宽度(300-600px)

**行为设置**:
- 点击图标行为(侧边栏/新标签/Zotero)
- DOI检测延迟(立即/2秒/5秒)
- 自动隐藏图标(永不/5分钟/10分钟)

**高级设置**:
- 自定义 Researchopia 服务器地址
- 启用调试日志
- 清除缓存

<a name="高效阅读技巧"></a>
## 17. 高效阅读技巧

### 17.1 分层阅读法

**第一遍:快速浏览**
1. 阅读标题、摘要、结论

2. 理解研究问题和方法
3. 创建标注记录关键信息
4. 查看他人对重点部分的标注

**第三遍:批判性阅读**
1. 评估论文的创新性和局限性
2. 与已有研究对比
3. 思考如何应用到自己的研究
4. 参与社区讨论

### 17.2 标注策略

**高亮重点**:
- 🔴 **红色**: 核心贡献和结论

- 🟢 **绿色**: 值得进一步探索的内容

**笔记模板**:


- 提出问题引发讨论

<a name="标注分享策略"></a>
## 18. 标注分享策略

### 18.1 公开 vs 私密

**何时选择公开**:
- ✅ 对论文的客观理解和总结
- ✅ 希望与社区交流的观点
- ✅ 发现的错误或疑问
- ✅ 对新手有帮助的解释

**何时选择私密**:
- 🔒 正在进行的研究想法
- 🔒 批判性评价(避免冒犯)
- 🔒 个人的todo和提醒
- 🔒 涉及未发表数据的讨论

### 18.2 撰写高质量标注

**好的标注示例**:
```
这里提出的"注意力机制"是本文的核心创新。与传统RNN不同，
模型可以直接关注输入序列的任意位置，解决了长距离依赖问题。
公式(3)中的权重计算值得仔细推导。

参考: Bahdanau et al. (2014) 首次提出了类似思想。
```

**避免的做法**:
- ❌ "这里很重要" (缺乏具体内容)
- ❌ "不懂" (没有具体问题)
- ❌ "作者是傻X" (不文明)

### 18.3 标注礼仪

- ✅ 尊重他人观点
- ✅ 建设性批评
- ✅ 引用时注明出处
- ✅ 及时回复讨论
- ❌ 不要剽窃他人的标注
- ❌ 不要发布无关内容

## 19. 隐私和安全

### 19.1 数据隐私

**我们收集的数据**:
- 账号信息(邮箱、用户名)
- 阅读历史(访问的论文)
- 标注内容(您创建的标注和评论)
- 社交互动(关注、点赞)

**我们不会**:
- ❌ 出售您的个人数据
- ❌ 与第三方分享未经许可的数据
- ❌ 追踪您在其他网站的行为

### 19.2 数据控制

**导出数据**:
1. 个人资料 → 设置 → 数据导出
2. 下载包含所有标注和历史的JSON文件

**删除数据**:
1. 删除单条标注: 直接在标注页面删除
2. 删除账号: 设置 → 账号 → 删除账号
3. 数据会在30天内完全删除

### 19.3 安全建议

- ✅ 使用强密码(至少8位,包含字母数字符号)
- ✅ 定期更换密码
- ✅ 不要与他人共享账号
- ✅ 退出时登出(尤其公共电脑)
- ⚠️ 警惕钓鱼邮件(Researchopia 不会索要密码)

## 20. 常见问题FAQ

### Q1: Researchopia 是免费的吗?

**A**: 是的! Researchopia 目前完全免费使用,包括:
- 无限制搜索和浏览论文
- 无限制创建标注
- 无限制共读会话
- 所有社交功能

未来可能推出高级功能(如团队协作、API访问),但核心功能将始终免费。

---

### Q2: 如何在 Researchopia 上找到论文全文?

**A**: Researchopia 不提供论文PDF托管,但会链接到:
- 期刊官网
- 开放获取仓库(如 arXiv, PubMed Central)
- 作者个人主页
- Sci-Hub (如果在您所在国家合法)

建议通过学校图书馆或 Google Scholar 访问。

---

### Q3: 我的标注会被其他人看到吗?

**A**: 取决于您的隐私设置:
- **公开**: 所有用户可见
- **仅好友**: 只有您关注的人可见
- **私密**: 只有您自己可见

默认设置为"公开",可在 设置 → 隐私 中修改。

---

### Q4: 如何邀请同事加入共读会话?

**A**: 
1. 创建会话时选择 "Private"
2. 复制生成的6位邀请码(如 `ABC123`)
3. 通过邮件、微信等发送给同事
4. 同事在 Zotero 插件中点击 "Join Session" 输入邀请码

**A**: 
- ✅ Zotero 7.0 及以上
- ✅ Zotero 8.0 Beta (推荐)
- ❌ Zotero 6.x 及以下(不支持)

### Q6: 浏览器扩展为什么没有检测到 DOI?

**A**: 可能的原因:

3. 网站 HTML 结构变化 → 提交 Issue 反馈
4. 扩展未正确加载 → 重新加载扩展
