# 账号注册和登录

欢迎使用 Researchopia！本页面将指导您完成账号注册和登录流程。

## 注册新账号

### 步骤 1: 访问注册页面

1. 访问 [www.researchopia.com](https://www.researchopia.com)
2. 点击右上角的 **"注册"** 按钮
3. 或直接访问 [注册页面](/auth/register)

### 步骤 2: 填写注册信息

在注册表单中填写以下信息：

- **邮箱地址**: 用于接收验证邮件和通知
- **密码**: 至少 8 位，建议包含字母和数字
- **用户名**: 显示在个人主页和评论中

### 步骤 3: 验证邮箱

1. 点击 **"创建账号"** 按钮
2. 查收邮箱中的验证邮件
3. 点击邮件中的验证链接激活账号
4. 返回登录页面

## 登录账号

### 基本登录

1. 点击右上角的 **"登录"** 按钮
2. 输入您的邮箱和密码
3. (可选) 勾选 **"记住我"** 保持登录状态
4. 点击 **"登录"**

### 忘记密码？

如果忘记了密码，请按以下步骤重置：

1. 点击登录框下方的 **"忘记密码"** 链接
2. 输入您的注册邮箱
3. 查收重置密码邮件
4. 点击邮件中的链接
5. 设置新密码

## 账号安全

### 密码要求

为了保护您的账号安全，密码必须满足以下要求：

- ✅ 至少 8 个字符
- ✅ 包含大小写字母
- ✅ 包含数字
- ✅ 推荐包含特殊字符 (!@#$%^&*)

### 安全建议

- 🔒 使用强密码，避免使用生日、姓名等容易被猜到的信息
- 🔄 定期更换密码（建议每 3-6 个月）
- 🚫 不要与他人共享账号
- 💻 在公共电脑上使用后记得退出登录

## 双因素认证（即将推出）

## 常见问题

### 为什么无法注册？

可能的原因：
- 邮箱已被注册（尝试使用"忘记密码"功能）
- 邮箱格式不正确
- 网络连接问题

### 如何更改邮箱？

目前需要联系支持团队更改邮箱。未来我们会在个人设置中添加此功能。

### 可以使用第三方账号登录吗？

我们计划支持以下第三方登录方式：
- ORCID
- Google Scholar
- GitHub
