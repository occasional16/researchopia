# 安装浏览器扩展

Researchopia 浏览器扩展让您在任何学术网页上快速访问论文信息。

## 支持的浏览器

- ✅ Google Chrome (v88+)
- ✅ Microsoft Edge (v88+)
- ✅ Brave
- ⏳ Firefox (即将支持)
- ⏳ Safari (计划中)

## 安装步骤 (Chrome)

### 方法 1: 从 Chrome Web Store

1. 访问 [Chrome Web Store](https://chrome.google.com/webstore)
2. 搜索 "Researchopia"
3. 点击 **"Add to Chrome"**
4. 点击 **"Add extension"** 确认

### 方法 2: 手动安装 (开发版)

1. 下载扩展源码 ZIP 文件
2. 解压到本地文件夹
3. 打开 Chrome → 更多工具 → 扩展程序
4. 开启 **"开发者模式"**
5. 点击 **"加载已解压的扩展程序"**
6. 选择解压后的文件夹

## 授予权限

首次使用需要授予以下权限：

- ✅ **访问网站数据**: 用于检测 DOI
- ✅ **存储**: 保存用户设置
- ✅ **活动标签**: 显示悬浮图标

## 验证安装

安装成功后：
- 浏览器工具栏出现 Researchopia 图标
- 访问学术网站时自动检测 DOI
